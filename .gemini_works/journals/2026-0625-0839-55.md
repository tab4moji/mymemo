## 2. 試行錯誤と決定事項
* パース後のHTMLの各見出し（H1〜H6）に対し、一意の ID （スラグ化ID）とアンカーリンクマーク（🔗）を自動付与するよう `markdown_parse.js` の `marked` レンダラーを拡張した。
* コピー可能であることを直感的に伝えるため、CSS にて `cursor: copy;` を設定し、ホバー時にコピーカーソルに変化するようにした。また、通常時はアンカーマークを非表示（`opacity: 0`）にし、見出しにホバーした時のみ表示されるスタイリッシュなUI設計にした。
* クリックされた際に、現在のページURLに該当見出しのフラグメント（例: `#見出し名`）を付加したショートカットURLをクリップボードにコピーするロジックを `markdown.js` に実装。コピー成功時に一時的（1.5秒間）に表示を ` Copied!` に変更し、ユーザーへの明確なフィードバックを提供するようにした。
* 各JSファイルに Code Stratification Rules に基づく Stratum Docstring（地層データ）を追加し、更新日時を `2026-06-25T08:39:55+09:00` に設定した。
