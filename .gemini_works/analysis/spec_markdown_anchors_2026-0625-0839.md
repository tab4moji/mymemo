### <core_logic> 1. アンカーIDの生成とスラグ化 </core_logic>
- **事実:** 見出しテキスト内の HTML タグを除去 (`text.replace(/<[^>]*>/g, '')`) した上で、トリム、小文字化、複数スペースのハイフン置換を行い、日本語、アルファベット、数字、ハイフン以外の記号を除去して ID 用文字列を生成。
- **推論:** 日本語を含む見出しでもブラウザのページ内遷移（フラグメントリンク）が正しく動作し、無効な文字によるID破壊を防ぐため。

### <core_logic> 2. カーソルの変更とコピー操作 </core_logic>
- **事実:** CSS で `.heading-anchor` に `cursor: copy;` を指定。ホバー時にコピーカーソルに変化する。
- **事実:** イベントデリゲーションを使用して、`#content` 要素でのクリック時に `.heading-anchor` をターゲット検知し、`window.location.href.split('#')[0] + anchorValue` の形式で URL を組み立て、`navigator.clipboard.writeText` でコピーを実行する。
